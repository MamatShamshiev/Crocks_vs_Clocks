Upd: в jupyter-notebook файле training_testing.ipynb исполнены все описанные ниже шаги.

Для запуска обучения свёрточной нейронной сети:
1) Загрузить изображения обучающей выборки (numpy array со значениями из [0..255] размера (N, C, H, W), где
N - длина обучающей выборки, C - количество каналов изображения, H - высота изображений, W - ширина изображения), а также
вектор меток классов.
2) С помощью функции loader_from_numpy() из модуля ConvNetwork.py создать "загрузчик" объектов выборки.
3) Создать объект класса ConvNetAutoEncoder из модуля ConvNetwork.py (например, 
в процессе решения задания были подобраны следующие параметры (см. solution.ipynb):
    input_size=(3, 32, 32), conv_layers_num=1, conv_out_channels=25, conv_kernel_size=4, conv_stride=2, pool_kernel_size=2, 
    pool_stride=1)
4) Обучить нейронную сеть с помощью функции fit_net() из модуля ConvNetwork.py.
